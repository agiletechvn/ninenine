Method: Constructor ( { canvasID, templateFill, templateFillOver } )

  • canvasID such as "main-canvas", return FFlivesEditorException exception if there is no canvas given
  • templateFill fill color of each rectangle
  • templateFillOver fill color of hover rectangle 
            $construction

Method: toJSON (   ) 

            $toJSON

Method: loadJSON ( jsonStr, callback )

  • jsonStr The json string
  • callback The callback function
            $loadJSON

Method: isGoingOut ( image, rect, padding = 20 )

This method checks whether an image object is going out of a rectangle

  • image a fabric.Image object
  • rect a fabric.Rect object
  • padding check is going out with a padding
                                $isGoingOut

Method: findByClipName ( name )

find a rectangle by a clip name

  • name clip name such as "rect_1"
                                $findByClipName

Method: findByClipFor ( name )

find an image belongs to a rect with a clip name

  • name clip name such as "rect_1"
                                $findByClipFor

Method: clipByName ( context, obj )

clip an image if it is moving out of a rect to which it belongs

  • context this is the context of a canvas
  • obj this is an image object
                                $clipByName

Method: findCNameByPosition ( x, y )

find a rect with a point

  • x left postion of the point
  • y top position of the point
                                $findCNameByPosition

Method: getAbsoluteCoords ( x, y )

return the current coordinates of the position relative to the editor

  • x left postion of the point
  • y top position of the point
                                $getAbsoluteCoords

Method: fitObject ( obj )

size your object but remain the ratio until the whole object fitting within your rect container

  • obj fabric.Object, such as fabric.Image
                                $fitObject
